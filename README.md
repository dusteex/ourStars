# Карта Созвездий Памяти

Интерактивная 3D карта созвездий, где каждая звезда представляет памятный день из совместной жизни.

## Технологии

- React 18
- TypeScript
- Three.js
- React Three Fiber
- Framer Motion

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Функциональность

- Просмотр созвездий в 3D пространстве
- Клик по звезде для просмотра деталей
- Модальное окно с описанием, галереей изображений и аудио
- Переключение между созвездиями через UI
- Плавные переходы камеры

## Структура данных

Данные о созвездиях и звездах находятся в `src/data/mockData.ts`. Вы можете добавить свои собственные воспоминания, изменив этот файл.
